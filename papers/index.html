Published papers

AI shaping papers: category definition and public capability evidence

The public release separates the simple category idea from bounded capability evidence: shape a practice-guided, subject-aware work pattern, carry source basis, domain-practice application, subject-context reasoning and work state, and keep human authority explicit.

Cold-reader guidance

Start with the lightest useful route

The papers are the formal DOI-backed release. They are best for category definition, public evidence review and citation. If you are new to the concept, use the lightest useful route first: start with the 5-minute page, then return here when you need the formal paper route. The website explains and routes; the papers remain the formal public record.

Release status and responsible use

Both papers are DOI-backed public preprints published on Zenodo. They are intended for public category understanding and bounded public-stage capability evaluation.

Use the stable DOI records and the paper landing pages for citation and latest-version access.

Use boundary

This boundary applies to both public papers: they support category understanding and bounded public-stage evidence evaluation, not implementation reliance.
