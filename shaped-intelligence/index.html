Concept page

Shaped intelligence

Shaped intelligence is reusable domain-facing work capability that carries a practice-guided, subject-aware work pattern and preserved work state while human direction, judgement and action remain explicit.

Plain-first guide

Think of AI that starts with the job’s work basis

Shaped intelligence is the reusable domain-facing capability. In plain terms, it means the AI contribution starts from a prepared work pattern and preserved work state rather than a fresh generic request.

This makes the page easier to read if you keep one question in mind: what burden no longer has to be rebuilt by the person each time the work resumes?

AI-shaping intelligenceBuilds and improves the pattern.
Shaped intelligenceCarries the pattern into recurring domain work.
Work pattern

It carries how the work should run

Shaped intelligence carries the role, sources, review boundaries and output pattern for a recurring work domain.

Work state

It carries where the work is up to

It preserves the current state of the work: prior decisions, open issues, risks, dependencies, review status and next action.

Definition

Shaped intelligence is the domain-facing reusable capability that carries more recurring work burden in a work domain, including domain-practice application and subject-context reasoning, while human direction, judgement and action remain explicit.

In plain English: it is AI working from a shaped, reusable work pattern for a specific kind of job, rather than from a fresh one-off prompt.

Why this term is needed

The value is continuity with practice context. A person should not need to rebuild the whole AI work basis every time the work resumes, changes hands or receives new information.

The practical difference from ordinary prompting is that the work basis continues, instead of being rebuilt whenever the work changes, pauses or changes hands.

How it differs from adjacent ideas

Adjacent ideaDistinction
Raw model capabilityRaw capability can answer broadly. Shaped intelligence is fitted to a domain pattern, source basis, review route, domain-practice guidance, subject-specific context and continuity burden.
A generated artefactA generated artefact is an output. Shaped intelligence is the reusable capability that helps carry recurring work burden across outputs.
Human replacementShaped intelligence carries burden, not responsibility. Human direction, judgement, approval, escalation and real-world action remain with the person.

Example

A domain-facing support capability that carries recurring reconstruction, sequencing, tracking, risk framing, resumption and communication-preparation work into review-ready form.

Non-example

A polished answer that does not remember the work state, unresolved decisions, review basis or next-action logic.

Assessment signal

The test is whether repeated work becomes easier to resume, review and continue from a preserved work state while preserving human accountability.
